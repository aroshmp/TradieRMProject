# Backend User Stories

Based on Business Requirements (BR1–BR10)

---

## BR2 – API / Webhook Integration with External Website

**US2.2 – Receive Job Requests via API/Webhook**
> As an **administrator**, I want the system to receive job requests from an external website via an API or webhook, so that new client requests are automatically captured and ready for processing.

---

## BR3 – Automatic Confirmation Response

**US3.1 – Automatic Acknowledgement to Client**
> As a **client**, I want the system to automatically send a confirmation message after a job request is submitted, so that the client knows the company has received their request without any manual intervention.

---

## BR4 – AI-Generated or Curated Response Suggestion

**US4.1 – Response Suggestion for Administrator**
> As an **administrator**, I want the system to generate or curate a suggested response to a client's request using AI, so that relevant and accurate reply options are available for review and sending.

**US4.2 – Response Suggestion for Technician**
> As a **technician**, I want the system to generate a suggested response to a client query, so that a professionally drafted message is ready for my review and approval.

---

## BR5 – Human Approval Before Sending AI Content

**US5.1 – Administrator Approval of AI Response**
> As an **administrator**, I want the system to hold all AI-generated responses in a pending state until I approve them, so that no AI content is sent to a client automatically without human review.

**US5.2 – Technician Approval of AI Response**
> As a **technician**, I want the system to require my explicit approval before sending any AI-generated message to a client, so that all outgoing communications are verified by a human before delivery.

---

## BR6 – Technician Timetable with Travel Time

**US6.2 – Auto-Adjust for Travel Time**
> As an **administrator**, I want the system to automatically calculate and account for travel time between jobs when scheduling a technician, so that technicians are not overbooked and can realistically complete each job on time.

---

## BR10 – Invoice Generation

**US10.1 – Generate Job Invoice**
> As an **administrator**, I want the system to automatically generate an invoice once a job is completed, detailing the labour costs and parts used, so that the client receives an accurate and professional billing document.

---

*Total Backend User Stories: 8*
